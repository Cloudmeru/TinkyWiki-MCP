Installation

Get TinkyWiki MCP running in under a minute.

Prerequisites

  • Python 3.10+
  • Playwright Chromium — installed automatically via playwright install chromium

Option A — PyPI (recommended)

The simplest way. Install from PyPI and you get the tinkywiki-mcp command globally:

pip install tinkywiki-mcp
playwright install chromium

Verify it works:

tinkywiki-mcp --help

Option B — From Source

Clone the repository and install:

git clone https://github.com/Cloudmeru/TinkyWiki-MCP.git
cd TinkyWiki-MCP
pip install .
playwright install chromium

For development (includes pytest and test dependencies):

pip install -e ".[test]"

Option C — Docker

# Build the image
docker build -t tinkywiki-mcp .

# Run with stdio (for MCP clients)
docker run -it --rm tinkywiki-mcp

# Run with SSE (for HTTP access)
docker run -p 3000:3000 tinkywiki-mcp --sse --port 3000

# With custom config
docker run -e TINKYWIKI_MAX_RETRIES=5 -e TINKYWIKI_VERBOSE=true tinkywiki-mcp

CLI Options

tinkywiki-mcp [--stdio | --sse] [--port PORT] [--verbose | -v]
FlagDescription
--stdioRun with stdio transport (default)
--sseRun with SSE transport
--port PORTPort for SSE transport (default: 3000)
--verbose, -vEnable debug logging
