Sample Conversation

The recording below shows a complete conversation using the TinkyWiki Agent (@tinkywiki) in the VS Code Chat panel. The agent uses TinkyWiki MCP tools to query Google TinkyWiki and deliver AI-generated documentation about open-source repositories.

What's Happening

In this demo, the user invokes @tinkywiki in the VS Code Chat panel. Here's the flow:

1

User sends a prompt

The user types a question like @tinkywiki What is facebook/react? in the VS Code Chat panel.

2

Master agent routes the request

The TinkyWiki Master orchestrator analyzes the intent and delegates to the appropriate specialist subagent (Researcher, Code Review, Architecture Explorer, Comparison, or Synthesizer).

3

Subagent calls MCP tools

The specialist subagent calls TinkyWiki MCP tools — tinkywiki_list_topics, tinkywiki_read_structure, tinkywiki_read_contents, or tinkywiki_search_wiki — to fetch real documentation from Google TinkyWiki.

4

TinkyWiki MCP fetches the data

The MCP server uses Playwright headless Chromium to render Google TinkyWiki's Angular SPA, then parses the content with BeautifulSoup and returns structured, citation-rich documentation.

5

Agent presents the result

The master agent presents the complete, synthesized response to the user — including tables, code snippets, architecture diagrams, and citations from TinkyWiki sections.

Features Demonstrated

Agent Orchestration

The master agent routes requests to the right specialist via the agent tool — no manual subagent selection needed.

MCP Tool Calls

Real TinkyWiki MCP tool invocations — list topics, read structure, read contents, search wiki — visible in the chat flow.

Keyword Resolution

Bare keywords like "react" are auto-resolved to facebook/react with interactive disambiguation when multiple matches are found.

Rich Responses

Responses include formatted markdown, code snippets, tables, and citations — all grounded in real Google TinkyWiki documentation.

Try It Yourself

Get started in under 2 minutes:

# Install
pip install tinkywiki-mcp
playwright install chromium

# Add to VS Code (.vscode/mcp.json)
{
  "servers": {
    "tinkywiki-mcp": {
      "type": "stdio",
      "command": "tinkywiki-mcp"
    }
  }
}

Then open the VS Code Chat panel (Ctrl+Alt+I), type @tinkywiki followed by your question, and watch the agent work.

Before running @tinkywiki, make sure the master agent, all 5 specialist sub-agents, and the skills file are properly configured in your .vscode/ folder. See the Agentic AI Guide for agent setup and Skills & Client Setup for the skills configuration.
