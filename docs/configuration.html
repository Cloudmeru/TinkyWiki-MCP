Configuration

All behaviour is configurable via environment variables — no code changes needed.

Environment Variables

Timeouts

VariableDefaultDescription
TINKYWIKI_HARD_TIMEOUT60Hard timeout per request (seconds)
TINKYWIKI_HTTPX_TIMEOUT30HTTP timeout fallback (seconds)
TINKYWIKI_PAGE_LOAD_TIMEOUT30Playwright page load timeout (seconds)
TINKYWIKI_ELEMENT_WAIT_TIMEOUT20Element wait timeout (seconds)
TINKYWIKI_RESPONSE_WAIT_TIMEOUT45Chat response wait timeout (seconds)

Retry & Limits

VariableDefaultDescription
TINKYWIKI_MAX_RETRIES2Max retry attempts
TINKYWIKI_RETRY_DELAY3Delay between retries (seconds)
TINKYWIKI_RESPONSE_MAX_CHARS30000Max response character count

Cache

VariableDefaultDescription
TINKYWIKI_CACHE_TTL300Rendered page cache TTL (seconds)

TINKYWIKI_SEARCH_CACHE_MAX_SIZE30Max search results in cache
TINKYWIKI_PARSED_CACHE_MAX_SIZE30Max parsed WikiPage objects in cache
TINKYWIKI_TOPIC_CACHE_TTL1800Topic list cache TTL (seconds) — longer since structure rarely changes
TINKYWIKI_TOPIC_CACHE_MAX_SIZE30Max topic lists in cache

Passing Environment Variables

Shell

TINKYWIKI_VERBOSE=true TINKYWIKI_CACHE_TTL=600 tinkywiki-mcp

Docker

docker run -e TINKYWIKI_MAX_RETRIES=5 -e TINKYWIKI_VERBOSE=true tinkywiki-mcp

VS Code MCP config

Pass environment variables via the env field in .vscode/mcp.json:

{
  "servers": {
    "tinkywikiMcp": {
      "type": "stdio",
      "command": "tinkywiki-mcp",
      "env": {
        "TINKYWIKI_VERBOSE": "true",
        "TINKYWIKI_CACHE_TTL": "600"
      }
    }
  }
}

Claude Desktop

Similarly, use the env key in claude_desktop_config.json:

{
  "mcpServers": {
    "tinkywiki": {
      "command": "tinkywiki-mcp",
      "env": {
        "TINKYWIKI_VERBOSE": "true"
      }
    }
  }
}
